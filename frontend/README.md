**CardioCare** is a modern, AI-powered healthcare application designed to assess cardiovascular disease risk using advanced machine learning models (Random Forest). The application features a production-grade UI/UX, fully responsive design, dark mode support, and interactive data visualizations.

## üöÄ Features

-   **AI Risk Prediction**: Instant cardiovascular risk assessment based on key health metrics.
-   **Modern Design**: Built with a "Medical Professional" aesthetic using **Tailwind CSS** and **shadcn/ui**.
-   **Dark/Light Mode**: Fully theme-aware UI with smooth toggling (using `next-themes`).
-   **Interactive Charts**: Visual breakdown of model accuracy and feature importance using **Recharts**.
-   **Responsive Layout**: Optimized for mobile, tablet, and desktop devices.
-   **Production Ready**: Type-safe codebase (TypeScript) with strict validation.

## üì¶ Getting Started

### Prerequisites
-   Node.js (v18+)
-   Python 3.9+ (for Backend)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/virajodedra/CardioCare.git
    cd CardioCare
    ```

2.  **Setup Frontend**:
    ```bash
    cd frontend
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to view the app.
